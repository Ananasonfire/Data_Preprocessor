## Краткое описание

Реализован класс `DataPreprocessor` для предварительной обработки табличных данных на основе `pandas.DataFrame`.

Решение представляет собой законченный и воспроизводимый pipeline обработки данных, включающий очистку, трансформацию и нормализацию признаков. Проект сопровождается модульными тестами и демонстрационным примером.

---

## Что реализовано

- Очистка данных:
  - удаление столбцов с большим количеством пропусков;
  - импьютация числовых и категориальных признаков.
- One-hot encoding категориальных признаков.
- Нормализация числовых данных (MinMax / Standard).
- Stateful pipeline:
  - сохранение параметров обучения;
  - корректная обработка новых данных.
- Валидация входных параметров и обработка ошибок.
- Полное unit-тестирование.
